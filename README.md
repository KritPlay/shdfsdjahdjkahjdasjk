# rcpsp-marketing

Учебный/исследовательский проект по RCPSP (Resource-Constrained Project Scheduling Problem)
с маркетинговыми эффектами (изменение спроса/дохода от смещения стартов задач‑услуг).

Структура проекта описана в `info.md`. Большинство модулей пока являются заглушками и
могут постепенно наполняться реализацией.
